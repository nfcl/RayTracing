# RayTracing

基于开源项目[Ray Tracing in One Weekend Book Series](https://github.com/RayTracing/raytracing.github.io)的《In One Weekend》搓的光线追踪渲染器

目前因为渲染过慢有线程池加速（[Main](https://github.com/nfcl/RayTracing/tree/main)）和不加速（[Base](https://github.com/nfcl/RayTracing/tree/base)）两个分支。

主要更改在不加速（Base）的分支上，然后将更改同步到加速分支上运行出图。

每个章节需要出图的地方都会在Release中上传版本。
